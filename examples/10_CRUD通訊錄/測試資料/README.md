## 🎯 練習情境

### 情境 1：批次匯入 CSV

```
我想把 sample_contacts.csv 的資料批次匯入到 Supabase。
步驟：
1. 讀取 CSV 檔案
2. 預覽要匯入的資料
3. 確認後寫入資料庫
```

### 情境 2：資料轉換

```
我想把 JSON 格式的聯絡人資料轉換成 CSV 格式，
然後再匯入到資料庫。
```

### 情境 3：資料清理

```
幫我檢查測試資料：
- 移除電話中的空格和特殊符號
- 統一 Email 格式為小寫
- 移除重複的聯絡人
```

### 情境 4：欄位對應

```
我有一份舊的通訊錄 CSV，欄位名稱不一樣：
- 「全名」→ 對應「name」
- 「手機號碼」→ 對應「phone」
- 「電子信箱」→ 對應「email」
幫我做欄位對應後匯入
```

## 💡 小技巧

### CSV 編碼問題

如果用 Excel 開啟 CSV 出現亂碼：
1. 用記事本開啟 CSV
2. 另存新檔，編碼選「UTF-8 with BOM」
3. 重新用 Excel 開啟

### 電話格式處理

Python 程式碼範例：
```python
# 移除電話中的符號，只保留數字
phone = phone.replace("-", "").replace(" ", "")
```

### 批次匯入時的注意事項

1. **先預覽再匯入**：確認資料正確再寫入資料庫
2. **加入錯誤處理**：如果某筆資料有問題，不要讓整個匯入失敗
3. **顯示進度**：大量資料匯入時，顯示目前進度
