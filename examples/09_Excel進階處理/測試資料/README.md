# 📊 Excel 進階處理 - 測試資料

## 📁 檔案清單

| 檔案 | 說明 |
|------|------|
| `attend.xlsx` | 出勤資料（含多個工作表） |

---

## 📋 attend.xlsx 工作表結構

| 工作表名稱 | 內容 | 資料量 |
|------------|------|--------|
| `員工名單` | 員工基本資料（含敏感個資） | 約 30 筆 |
| `計算0901` ~ `計算1028` | 每日出勤計算資料 | 20 個工作表 |

---

## 🎯 練習情境

### 情境 1：敏感資料匿名化

將姓名和身分證字號替換為亂數生成的假資料，同時保持關聯性。

```
幫我處理 attend.xlsx，把所有工作表中的敏感資料做匿名化：
- 姓名改成亂數中文假名
- 身分證字號改成亂數假號碼
- 同一個人的假資料要一致
```

---

### 情境 2：合併多個工作表

將所有「計算」開頭的工作表垂直合併成一個。

```
幫我把 attend.xlsx 裡所有「計算」開頭的工作表合併成一個，
加上「來源工作表」欄位來標註資料來自哪個工作表
```

---

## ⚠️ 注意事項

- 處理敏感資料前，請先備份原始檔案
- 匿名化後的對照表應妥善保管，避免外洩
- CSV 匯出建議使用 UTF-8 with BOM 編碼
